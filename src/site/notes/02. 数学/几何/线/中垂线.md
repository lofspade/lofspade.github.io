# 定义
中垂线（又称垂直平分线）是指经过某一条线段的中点，并且垂直于这条线段的直线。它需同时满足两个条件：

1. ​**过线段中点**​：直线必须通过线段的中点；
2. ​**垂直于线段**​：直线与线段的夹角为90°

例如，若线段为 $AB$，其中点为$N$，过 $N$ 作 $MN⊥AB$，则 $MN$ 即为 $AB$ 的中垂线

---

# 定理

> [!warning] 性质定理
> **中垂线上任意一点，到线段两端点的距离相等**​
{ #80cc35}


- 符号表示：若 $MN$ 是 $AB$ 的[[02. 数学/几何/线/中垂线\|中垂线]]，$P$ 是 $MN$ 上任意一点，则 $PA=PB$。
- ​**证明思路**​：通过[[02. 数学/几何/几何关系/轴对称\|02. 数学/几何/几何关系/轴对称]]性（沿 $MN$ 折叠，点 $A$ 与 $B$ 重合）或全等三角形（如 $△PAN≅△PBN$）可证。
---

> [!warning] 逆定理(判定定理)
> **到线段两端点距离相等的点，必在该线段的中垂线上**​

- 即若 $PA=PB$，则点 $P$ 在 $AB$ 的中垂线上。
- ​**证明思路**​：
    - 当 $P$ 在线段 $AB$ 上时，$P$ 为中点，显然在中垂线上；
    - 当 $P$ 在线段外时，$△PAB$ 为等腰三角形，作底边高 $PC$ 必为中垂线（三线合一）。
---
> [!warning] 三角形外心定理
> ​**三角形三条边的中垂线交于一点（外心），且该点到三个顶点的距离相等**​
- 外心是三角形外接圆的圆心，满足 $OA=OB=OC$（$O$ 为外心）。
- ​**应用举例**​：已知 $△ABC$，作 $AB$、$BC$ 的中垂线交于点 $O$，则 $O$ 也在 $AC$ 的中垂线上。

## 中垂线的作图方法（[[02. 数学/辅助技巧/尺规作图\|尺规作图]]）

1. 分别以线段 $AB$ 的端点 $A$、$B$ 为圆心，​**大于 $AB/2$ 的长度**为半径画弧；
2. 两弧在线段同侧（或两侧）相交于两点 $M$、$N$；
3. 连接 $M$、$N$，直线 $MN$ 即为 $AB$ 的中垂线。  

原理​：**利用等腰三角形底边的高与中线重合的性质。**

---

## 🏫 实际应用

- ​**选址问题**​：已知不在同一直线上的三个村庄 $A$、$B$、$C$，求建学校的位置（要求到三村距离相等）。  
    ​**解法**​：作 $AB$、$BC$ 的中垂线交于点 $O$，则 $OA=OB=OC$，$O$ 即为学校位置。

## 💎 关键总结

| ​**概念**​    | ​**核心内容**​                           |
| ----------- | ------------------------------------ |
| ​**定义**​    | 过线段中点且垂直于线段的直线。                      |
| ​**性质定理**​  | 中垂线上的点到线段两端距离相等 → 用于证明线段相等（如 PA=PB）。 |
| ​**逆定理**​   | 到线段两端距离相等的点在中垂线上 → 用于证明垂直平分关系。       |
| ​**三角形外心**​ | 三边中垂线共点且到顶点等距 → 解外接圆问题。              |

中垂线是[[02. 数学/几何/几何关系/轴对称\|02. 数学/几何/几何关系/轴对称]]图形对称轴的基础（如对称图形中任意两对称点的连线被对称轴垂直平分）。
