### 📌 一、不等式的定义与基本概念

1. ​**定义**​  
    用不等号（>，<，≥，≤，≠）连接两个解析式形成的式子称为不等式。例如：$x + 4 > 7$，$3x \leq 4 - x$。
    
    - ​**严格不等式**​：使用 > 或 <（如 $a > b$)。
    - ​**非严格不等式**​：使用 ≥ 或 ≤（如 $a \geq b$)。
2. ​**解与解集**​
    
    - ​**解**​：使不等式成立的未知数值（如 $x > 3$ 是 $2x > 6$ 的解）。
    - ​**解集**​：所有解的集合，通常在数轴上表示（如 $x \in [-1, 1]$) 。
3. ​**同解不等式**​  
    两个解集相同的不等式互为同解不等式（如 $x > 3$ 与 $2x > 6$ 同解）。

### ⚖️ 二、不等式的基本性质

以下性质是变形与证明的核心依据：

|​**性质名称**​|​**数学表达**​|​**文字描述**​|
|---|---|---|
|对称性|$a > b \Leftrightarrow b < a$|不等号方向可互换|
|传递性|$a > b, \, b > c \Rightarrow a > c$|大小关系可传递|
|可加性|$a > b \Rightarrow a \pm c > b \pm c$|加减同数不等号方向不变|
|乘正保号|$a > b, \, c > 0 \Rightarrow ac > bc$|乘除正数不等号方向不变|
|乘负变号|$a > b, \, c < 0 \Rightarrow ac < bc$|乘除负数不等号方向反转|
|同向可加|$a > b, \, c > d \Rightarrow a + c > b + d$|同向不等式可相加|
|同向正数可乘|$a > b > 0, \, c > d > 0 \Rightarrow ac > bd$|同向正不等式可相乘。|

​**易错点**​：

- 乘除含参变量时需讨论符号（如 $ac > bc$ 成立需 $c > 0$)。
- 倒数性质：若 $a > b > 0$，则 $\frac{1}{a} < \frac{1}{b}$ 。

### 📐 三、不等式的分类与解法

#### 1. ​**一元一次不等式**​

- ​**标准形式**​：$ax + b > 0$（$a \neq 0$)。
- ​**解法**​：移项、系数化1（注意乘负数时变号）。  
    ​**示例**​：$3 - x > 0 \Rightarrow x < 3$。

#### 2. ​**一元二次不等式**​

- ​**标准形式**​：$ax^2 + bx + c > 0$（$a > 0$)。
- ​**解法**​：
    - ​**代数法**​：因式分解后利用口诀“大于取两边，小于取中间”（如 $(x-2)(x-3) > 0 \Rightarrow x < 2$或 $x > 3$)。
    - ​**图象法**​：结合二次函数图像（开口向上时，解集为两根之外）。

#### 3. ​**高次不等式（穿根法）​**​

- ​**步骤**​：  
    ① 因式分解，根按序标在数轴；  
    ② 从右上方起穿根（奇次根穿过，偶次根反弹）；  
    ③ 根据不等式符号取区间（如 $(x+1)(x-2)^2 > 0 \Rightarrow x \in (-1, 2) \cup (2, +\infty)$) 。

#### 4. ​**不等式组**​

- ​**解集规则**​：
    - ​**同小取小**​：$x < a$ 且 $x < b \Rightarrow x < \min(a,b)$。
    - ​**相交取中**​：$a < x < b$（如 $x > -5$ 且 $x < 3$)。
    - ​**向背取空**​：解集无交集（如 $x > 1$ 且 $x < 0$)。

### 💡 四、不等式的应用

#### 1. ​**实际建模**​

- ​**优化问题**​：如生产成本最小化、利润最大化。  
    ​**案例**​：生产甲、乙产品，原料约束下求利润最大（线性规划）：
    - 目标函数：$z = 30x + 40y$；
    - 约束：$3x + 2y \leq 1200$，$x + 2y \leq 800$。
- ​**资源分配**​：如运输调度（仓库调货至商店运费最小化）。

#### 2. ​**函数与方程**​

- ​**恒成立问题**​：
    - $x \in \mathbb{R}$ 时 $f(x) \geq a$ 恒成立 $\Leftrightarrow$ 二次函数最小值 $\geq a$（或 $\Delta \leq 0$)。  
        ​**示例**​：$f(x) = x^2 + ax + 3 \geq a$ 恒成立 $\Rightarrow a \in [-6, 2]$。
- ​**参数范围**​：
    - 方程根分布（如 $x^2 + (m-3)x + m = 0$ 两根在 $(0,2)$ 内 $\Rightarrow m \in (1,3)$）。

#### 3. ​**比较大小与证明**​

- ​**作差法**​：$a - b > 0 \Rightarrow a > b$（如比较 $\frac{\ln 2}{2}$ 与 $\frac{\ln 3}{3}$）。
- ​**作商法**​：$\frac{a}{b} > 1 \Rightarrow a > b$（需 $a,b > 0$)。

### 🔍 五、解题技巧与注意事项

1. ​**分类讨论**​：含参不等式需按参数符号、根的大小讨论（如解 $2x^2 + ax + 2 > 0$，需分 $\Delta >0$、$\Delta=0$、$\Delta<0$)。
2. ​**数形结合**​：
    - 线性规划中目标函数几何意义（如 $z = ax + by$ 的截距最值）。
    - 二次不等式借助函数图像分析解集。
3. ​**转化与化归**​：
    - 多元问题转化为一元（如 $f(-2) = f(1) + 3f(-1)$ 求范围）。
4. ​**实际意义验证**​：解集需符合实际定义域（如时间、长度非负）。
